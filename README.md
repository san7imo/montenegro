# Montenegro Salud y Belleza

Landing page construida con `React + TypeScript + Vite`, alineada al manual de marca y preparada para crecer a futuro con servicios, reservas y contacto real.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Edición rápida

Todo el contenido editable principal está centralizado en:

- `src/content/siteContent.ts`

Los assets optimizados usados por la web están en:

- `public/assets`

## Notas

- El formulario y los datos de contacto quedaron preparados como placeholders porque todavía no existen datos finales confirmados.
- Se usaron las tres imágenes disponibles en el repositorio como soporte visual de la identidad.
